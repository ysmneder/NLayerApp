# NLayerApp
NLayer based Web Api Project is written using .Net 7

##Project Layout
- App.Api
    - Controllers
    - Filter
    - Middlewares
    - Modules
- App.Core
    - Dtos
    - Models
    - Repositories
    - Services
    - UnitOfWork
- App.Repository
    - Configurations
    - Migrations
    - Repositories
    - Seeds
    - UnitOfWork
- App.Services
    - Exceptions
    - Mapping
    - Services
    - Validations



## Technologies
- .NET 7
- Swagger (Documentation)
- Entity Framework Core
- Npgsql 
- AutoMapper
- FluentValidation
- AutoFac
- Docker-Compose
